# Keyboard shortcuts

## Global
| Pattern | Action |
| --: | :-: |
| `/` | Search |
| `?` | View Help |
| `.` | Edit in Gitpod |
| `Alt`/`⌘` + `↑` | Navigate to parent directory |

## Navigation
These shortcuts are only available when browsing a directory listing.

| Pattern | Action |
| --: | :-: |
| `↑` | Select above directory |
| `↓` | Select below directory |
| `Enter` | Enter directory |
